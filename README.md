## Featured Project

### CNS Paper Reader Skill

一个面向科研初学者的 CNS / 高分论文精读工具。

它可以把一篇复杂论文自动整理成完整中文文档，包括：

- Figure 逐图逐 panel 讲解
- 实验原理解释
- 结论—证据链分析
- 实验复现路线
- 组会汇报稿
- 一页纸总结

👉 [查看项目仓库](https://github.com/wwwcccsj-design/cns-paper-reader-v1.2-figure-doc.skill.git)
