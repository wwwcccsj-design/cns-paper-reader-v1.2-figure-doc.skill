# 实验复现路线模板

## 使用场景

当用户说“我要复现 / 准备复现 / 下一步怎么做 / 需要准备什么 / 给我实验方案”时使用。

---

## 复现设计原则

1. 先复现最核心、最容易验证的现象，不要一上来复现整篇论文。
2. 优先选择有明确 readout 的实验，如 PCR、Western blot、荧光、表型测定。
3. 先做最小可行复现版本（MVR），再扩展到机制链。
4. 必须列出对照组。
5. 必须区分论文给出的参数和需要用户确认的参数。
6. 不编造精确浓度、时间、温度；原文未给出时写“待查 Methods/SOP”。
