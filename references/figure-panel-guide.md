# Figure 逐图逐 Panel 精讲模板（带图、具体、不可笼统版）

## 使用场景

当用户要求“解读 figure / 每一张图 / 每个子图 / 图中逻辑 / 结构图解释 / 这张图怎么看 / 文档带 figure 图”时使用。

## 最高原则

Figure 解读不是写“这张图说明了什么”，而是把研究者的实验行为讲清楚：

> **研究者做了什么 → 怎么做的 → 为什么这样做 → 得到什么结果 → 这个结果说明什么 → 对全文有什么意义 → 证据有什么风险**

最终 Word 文档中，Figure 图片/截图必须尽可能出现在对应解读之前。

---

## 带图整图解读模板

```markdown
## Figure X： [图题中文概括]

[插入 Figure X 原图 / 页面截图 / 用户上传图片]

**图片来源**：论文 Figure X / PDF 第 X 页 / 用户上传截图。

**这张图在全文中的作用**：
[建立核心现象 / 鉴定关键分子 / 证明因果关系 / 解释机制 / 体内验证 / 临床相关性 / 最终模型]

**这张图想回答的核心问题**：
[一句话说明问题]

**研究者总体做了什么**：
[用 2–4 句话说明作者在这一整张图里做了哪些实验组合]

**他们为什么这样安排实验顺序**：
[解释从现象到机制、从相关到因果、从体外到体内的逻辑]

**主要方法组合**：
- [方法1（English term）]：用来回答[问题]
- [方法2（English term）]：用来验证[结论/排除替代解释]
- [方法3（English term）]：用来做[因果/机制/体内验证]

**整图结论**：
[这一整张图最终支撑的结论]
```

## 单个 Panel 强制模板

每个 panel 必须按下面 8 项输出；不允许只写一段空泛总结。

```markdown
### Figure Xa

| 解读项目 | 具体内容 |
|---|---|
| 1. 这个 panel 想回答什么问题 | [这个小图对应的科学问题] |
| 2. 研究者做了什么 | [具体实验/分析/比较/处理] |
| 3. 他们怎么做的 | [样本/模型/分组/处理/检测指标/数据分析方法] |
| 4. 为什么要这样做 | [说明这个实验为什么能回答上面的问题，是否用于建立现象、验证机制或证明因果] |
| 5. 图应该怎么看 | [横轴、纵轴、颜色、分组、误差棒、显著性、比例尺、门控、条带、代表图等] |
| 6. 得到什么结果 | [具体描述趋势、差异、变化方向；能引用数值就引用数值] |
| 7. 结果说明什么意义 | [该结果支持了作者哪一个小结论，对整篇论文主线有什么作用] |
| 8. 证据风险/局限 | [样本量、对照、代表图是否有定量、是否只能说明相关、是否还缺 rescue/功能验证] |

**初学者大白话解释**：
[用 2–3 句把这个 panel 讲给外行听。]
```

## 不同 Figure 类型的额外要求

### 1. 机制示意图 / 模型图 / Graphical model

必须逐元素解释：

- 每个细胞、蛋白、基因、代谢物、通路模块代表什么。
- 每个箭头表示激活、抑制、结合、运输、分泌、转录、降解还是实验流程。
- 颜色、形状、空间位置是否代表不同组、不同状态或不同细胞区室。
- 哪些部分有实验数据直接支持，哪些是作者根据数据推断出的模型。
- 这个模型对应哪些 Figure 的证据。

### 2. 实验流程图 / 技术路线图

必须讲清：

- 输入材料是什么：细胞、动物、样本、菌株、组织、数据库。
- 处理过程是什么：刺激、敲除、过表达、感染、给药、培养、筛选。
- 输出检测是什么：成像、测序、蛋白检测、功能实验、统计分析。
- 这个流程为什么能回答论文的主问题。

### 3. 柱状图 / 折线图 / 箱线图 / 散点图

必须讲清：

- 横轴是什么分组或时间点。
- 纵轴是什么指标，单位是什么。
- 每个点是否代表独立样本。
- n 值是多少，误差棒是 SD、SEM 还是 CI。
- 使用了什么统计检验。
- Y 轴是否截断，是否可能夸大差异。

### 4. 显微图 / 免疫荧光图（Immunofluorescence）

必须讲清：

- 每个通道代表什么标记。
- 合并图说明什么共定位或空间关系。
- 比例尺（scale bar）是多少。
- 是否只是代表性图片，是否有定量统计。
- 曝光、阈值、取景区域是否可能影响结论。

### 5. Western blot / Co-IP / Pull-down

必须讲清：

- 检测的蛋白是什么，内参/输入对照是什么。
- 条带增强或减弱代表什么。
- 是否有灰度定量和生物学重复。
- Co-IP/Pull-down 只能证明结合/相互作用证据，不能单独证明功能因果。

### 6. qPCR / RNA-seq / scRNA-seq / 空间组学

必须讲清：

- 样本来源和分组。
- 测的是基因表达、细胞类型、通路富集还是空间位置。
- 差异筛选标准是什么。
- 是否有独立实验验证关键基因。
- 测序结果通常是相关证据，需要功能实验进一步验证。

### 7. 流式细胞术（Flow cytometry）

必须讲清：

- 门控策略（gating strategy）。
- 横纵轴分别是什么 marker。
- 每个象限代表什么细胞群。
- 是否有 FMO、isotype、single-stain 对照。
- 代表图是否有统计汇总。
