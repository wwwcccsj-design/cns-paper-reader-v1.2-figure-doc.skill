# 第一轮扫读追问清单

执行第一轮快速扫读时，用此清单逐项确认，确保不遗漏关键信息。

---

## 摘要追问清单

阅读摘要时，需要能回答以下问题：

- [ ] **研究问题**：这篇论文要解决的核心科学问题是什么？（一句话）
- [ ] **知识缺口**：在此之前，领域内缺少什么？
- [ ] **核心方法**：用了什么主要技术或实验体系？
- [ ] **主要结论**：最重要的发现是什么？（不超过两点）
- [ ] **声称贡献**：作者认为这项工作的意义是什么？

**红旗警示**：若摘要中结论措辞模糊（如"我们的结果表明可能……"），第二轮需要重点核查数据是否支撑。

---

## 图表速览追问清单

浏览所有图表（含 Extended Data / Supplementary Figures）时确认：

- [ ] **主图数量**：共几张主图？几张补充图？
- [ ] **图表类型**：有哪些实验类型（Western blot / RNA-seq / 行为学 / 结构解析 / 临床数据……）？
- [ ] **叙事逻辑**：Figure 1 建立了什么？Figure 2 回答了什么问题？以此类推……
- [ ] **核心图**：哪一张图是整篇论文最重要的数据图？
- [ ] **统计标注**：图中是否有误差棒（error bar）和显著性标注（* / p值）？
- [ ] **样本量**：能否从图注中看到 n 值？

**判断标准**：如果看完所有图表后，能用一句话概括"论文用了什么方法证明了什么"，则第一轮图表阅读合格。

---

## 引言末段追问清单

引言最后 1–2 段通常包含：

- [ ] **研究假说**：作者明确提出的假说是什么？（找"We hypothesize..." / "Here we show..." / "We report..."）
- [ ] **论文结构**：作者是否预告了论文的组织方式？
- [ ] **范围声明**：这项研究的边界在哪里（体内/体外？细胞系/动物模型/人体？）

---

## 结论/讨论首段追问清单

- [ ] **作者自评**：作者认为最重要的 1–2 个发现是什么？
- [ ] **与现有知识的关系**：作者声称这项工作改变/拓展/验证了什么已有认识？
- [ ] **局限性预告**：作者是否在首段就承认了局限性？（好的论文通常会）

---

## 第一轮价值判断标准

完成扫读后，用以下标准决定是否值得继续深读：

| 判断维度 | 权重 | 评分（1–5） |
|---------|------|-----------|
| 研究问题与我的方向的相关性 | 高 | |
| 期刊级别与发表年份（时效性） | 中 | |
| 方法是否是我熟悉或需要学习的 | 中 | |
| 结论是否出乎意料或有启发性 | 高 | |
| 数据量和工作规模 | 低 | |

**建议阈值**：加权总分 ≥ 15 分（满分 25 分）才值得进入第二轮精读。
