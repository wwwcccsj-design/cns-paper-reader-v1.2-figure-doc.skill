# 第三轮批判性评估框架

执行第三轮批判性深读时，按此框架逐维度评估，输出批判性评估报告。

---

## 维度一：创新性评估

### 评判标准

| 等级 | 描述 | 典型特征 |
|------|------|---------|
| **范式转变** | 颠覆现有认知框架 | 发现全新机制、建立新领域、推翻主流观点 |
| **重大推进** | 显著填补知识缺口 | 首次实现 X、首次证明 Y、解决长期争议 |
| **方法创新** | 提供新工具/新技术 | 开发新方法，本身应用价值高 |
| **增量贡献** | 已有知识的扩展 | 在新细胞系 / 新疾病模型中验证已知机制 |
| **重复验证** | 证实已有结论 | 用不同方法重复他人结果 |

**追问**：
- 引言中的"knowledge gap"是真实存在的，还是被人为放大的？
- 这项发现 5 年前是否已有人做过类似工作？（关注作者是否引用了竞争性工作）
- 结论是否真的是 "first time"，还是 "first time in this specific context"？

## 维度二：数据质量评估

### 统计问题检查清单

- [ ] **样本量**：每组 n 值是否足够？（生物学实验通常需要 n ≥ 3 独立重复；动物实验 n ≥ 5–10；临床研究需要功效分析）
- [ ] **统计检验**：所用检验方法是否匹配数据类型（正态 vs 非参数？配对 vs 独立？）
- [ ] **多重比较**：比较了多个组/条件时，是否进行了多重检验校正？
- [ ] **误差棒**：是 SEM（偏小，视觉上好看）还是 SD（真实反映数据离散度）？
- [ ] **精确 p 值**：是否报告了精确 p 值，而不仅仅是 *、**、***？

### 实验设计问题检查清单

- [ ] **阴性对照**：关键实验是否有合适的阴性对照（vehicle control / scramble siRNA / isotype antibody）？
- [ ] **盲法**：主观性评分实验（病理评分、行为学）是否采用了盲法？
- [ ] **随机化**：动物分组是否随机化？
- [ ] **独立重复**：多少次生物学独立重复（biological replicates）？技术重复（technical replicates）不能替代生物学重复。
- [ ] **代表性**：Western blot / 免疫荧光等图像是否注明"representative of N experiments"？

### 数据可视化问题

- [ ] 图表是否选择性展示了最好的结果（cherry-picking）？
- [ ] Y 轴起点是否从 0 开始？（如不从 0 开始，差异可能被视觉放大）
- [ ] 散点图中的数据点是否遮挡过多？（boxplot 不能替代原始数据展示）

## 维度三：逻辑漏洞检查

### 常见逻辑问题

**1. 相关 ≠ 因果**
- 观察到 A 和 B 同时升高/降低 ≠ A 导致 B
- 追问：有没有功能性实验（gain-of-function / loss-of-function）来证明因果？

**2. 体外 ≠ 体内**
- 细胞系实验的结论能推广到动物模型或人体吗？
- 追问：有没有 in vivo 验证？使用的细胞系是否能代表生理状态？

**3. 过度概括**
- 在一种细胞类型 / 一种疾病 / 一个物种中发现的结果，能推广到所有情况吗？
- 追问：作者是否在结论中声称了超出数据范围的普遍性？

**4. 模型假设**
- 计算模型 / 预测模型是否经过了足够的实验验证？
- 追问：模型中有哪些关键假设？这些假设是否在本研究中被检验过？

**5. 发表偏倚**
- 阴性结果是否被选择性省略？
- 追问：补充材料中是否有与主图结论不一致的数据？

## 维度四：研究机会识别

### 系统性追问

**从局限性出发**：
- 作者承认的每一条局限，背后是否有可行的研究问题？
- "Future studies are needed to..." 后面提到的具体方向，是否有人在做？

**从知识缺口出发**：
- 本研究填补了 A 缺口，但引出了哪些新的 B 缺口？
- 引言中提到但本研究没有解决的问题是什么？

**从方法角度**：
- 本研究的技术手段是否可以迁移到你感兴趣的模型/疾病/问题上？
- 有没有更好的技术可以做这个研究（单细胞分辨率、空间转录组、CRISPR 筛选……）？

**从结论反向**：
- 如果这个结论是对的，它预测了什么？这些预测是否被验证过？
