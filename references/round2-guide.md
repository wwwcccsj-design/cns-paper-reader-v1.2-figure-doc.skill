# 第二轮结构化精读指引

执行第二轮精读时，按此指引逐节推进，最终输出结构化中文笔记。

---

## 各章节精读要点

### 引言（Introduction）

**目标**：还原作者构建的逻辑链，理解为什么这个问题值得研究。

精读时提取以下四层结构：

1. **领域背景**（通常第 1–2 段）：这个领域研究什么？目前达到了什么水平？
2. **已有工作**（通常第 2–3 段）：前人做了什么？用什么方法回答了哪些问题？
3. **知识缺口**（关键段落，通常含 "However..." / "Despite..." / "Remains unclear..."）：还有什么问题没解决？为什么没解决？
4. **本研究**（末段）：作者如何填补这个缺口？用什么策略？

**注意**：引言中引用的关键文献（作者通常会在此引用 1–3 篇奠基性论文）值得记录下来，它们是理解本文的背景基础。

---

### 结果（Results）

**目标**：理解每个实验的逻辑，确认数据是否支撑结论。

**逐段精读框架**（每个结果小节重复以下步骤）：

```
[小节标题]
问题：这个实验要回答什么问题？
方法：用了什么实验手段？关键参数是什么？
数据：主要数据是什么（定量值、趋势、显著性）？
结论：作者得出了什么结论？
图对应：对应哪张图/图注中哪个面板？
自我评价：数据是否清晰支撑结论？有无过度解读？
```

**图文对照检查点**：
- 正文描述的数据数值是否与图中一致？
- 作者用"significantly"时，p 值是多少？是否真的显著？
- 误差棒类型（SEM vs SD）是否在图注中注明？
- 代表性图像（representative image）后面是否有定量数据支撑？

---

### 讨论（Discussion）

**目标**：理解作者的解读，识别论文在领域内的定位。

精读时关注：

1. **结论重申**：作者如何用一段话总结全篇发现？（与摘要对比，是否一致？）
2. **与已有文献比较**：作者声称与哪些文献结果一致/不一致？给出了解释吗？
3. **机制解读**：作者提出了什么机制模型？（通常有 model figure）
4. **局限性声明**：作者承认了哪些不足？用词是否诚实（"limitation" vs "future study"）？
5. **未来方向**：提出了哪些悬而未决的问题？

---

### 方法（Methods / Materials and Methods）

**选读策略**：不需要逐字阅读，按需查阅。

| 需要查阅的情况 | 关注内容 |
|-------------|---------|
| 结果有疑问 | 该实验的具体操作步骤和参数 |
| 统计方法存疑 | 统计检验类型、样本量、重复次数 |
| 想重复该实验 | 完整方法细节 |
| 评估数据可靠性 | 盲法设计、随机化、阴性对照 |

**统计方法重点核查**：
- 使用了哪种统计检验（t-test / ANOVA / Mann-Whitney / Fisher's exact…）？
- 是否符合数据分布假设（正态分布 vs 非参数数据）？
- 多重比较是否做了校正（Bonferroni / FDR）？
- 动物实验：每组 n 值是多少？是否足够？

---

### 补充材料（Supplementary / Extended Data）

**重点查阅内容**：

- **控制实验（Control experiments）**：验证特异性的阴性对照、同工型对照
- **重复实验**：生物学重复（biological replicates）次数
- **原始数据**：是否提供了 source data（原始 blot、流式散点图等）
- **统计汇总表**：详细的统计数据（均值、SD、精确 p 值）

---

## 结构化中文笔记模板

完成第二轮精读后，输出以下格式的笔记：

---

```markdown

**标题**：[论文英文标题]
**期刊**：[期刊名] [年份] [卷/期]
**通讯作者**：[姓名]，[机构]
**阅读日期**：[日期]
**论文类型**：原创研究 / 综述 / 方法学 / 快报

---

[1–2 句，说明前人没有解决的问题]

- [作者, 年份, 期刊]：[一句话说明该文献的贡献]
```
