# 实验原理解释模板

## 使用场景

当用户问“这个实验为什么这么做 / 原理是什么 / 结果怎么看 / 这个方法能说明什么 / 我以前没做过”时使用。

---

## 输出原则

- 不只解释技术名，要解释“它为什么能回答论文问题”。
- 必须区分：检测相关性、证明因果、验证机制、排除替代解释。
- 每种方法都要说明需要什么对照。
- 初学者优先使用生活化类比，但不能牺牲科学准确性。

---

## 单个实验讲解模板

```markdown
### [实验名称（English term）]

**一句话作用**：
[这个实验在论文里负责证明什么]

**它想回答的问题**：
[科学问题]

**基本原理**：
[用初学者能懂的话解释原理]

**本文具体做法**：
- 样本/模型：[细胞、动物、组织、患者样本等]
- 处理条件：[药物、基因敲除、诱导、时间点等]
- 检测指标：[蛋白、RNA、荧光、表型等]
- 对照组：[阴性/阳性/空载体/野生型等]

**结果应该怎么看**：
- 如果结果升高/增强，说明：[解释]
- 如果结果降低/减弱，说明：[解释]
- 如果没有变化，说明：[可能解释]

**它能支持什么结论**：
[能证明的范围]

**它不能单独证明什么**：
[不能证明因果/不能证明体内真实发生等]

**关键对照**：
- [对照1]：为什么需要
- [对照2]：为什么需要

**常见失败原因**：
- [原因1]
- [原因2]
```

## 常见实验方法解释要点

### PCR / qPCR

- PCR 用于扩增特定 DNA 片段。
- qPCR 用于定量 RNA/cDNA 或 DNA 的相对丰度。
- 需要内参基因（housekeeping gene）和阴性对照。
- 只能说明表达量变化，不能直接证明蛋白功能变化。

### Western blot

- 用抗体检测目标蛋白大小和丰度。
- 条带位置对应分子量，条带深浅反映相对含量。
- 必须有 loading control，如 GAPDH、β-actin、Tubulin。
- 单张代表图需要配灰度定量和重复实验。

### 免疫荧光 / 共聚焦显微镜

- 用荧光抗体定位蛋白在细胞/组织中的空间分布。
- 共定位只能提示可能相互接近，不等于直接相互作用。
- 需要阴性抗体对照、单通道对照和定量分析。

### 流式细胞术

- 用荧光标记区分不同细胞群或状态。
- 关键是门控策略和 marker 组合。
- 需要补偿、FMO、isotype 或未染色对照。

### RNA-seq / scRNA-seq

- RNA-seq 反映群体平均转录水平。
- 单细胞 RNA-seq 反映细胞异质性。
- 差异表达结果通常需要 qPCR、Western blot 或功能实验验证。

### CRISPR / RNAi / 过表达

- 用于扰动基因功能。
- loss-of-function 能测试“缺失后会怎样”。
- gain-of-function 能测试“增强后会怎样”。
- rescue experiment 是证明特异性和因果关系的关键。

### Co-IP / pull-down / proximity labeling

- 用于检测蛋白相互作用或近邻关系。
- Co-IP 证明同一复合物中共沉淀，不一定是直接结合。
- 需要 IgG 对照、input、反向 Co-IP 或突变体验证。

### 动物模型实验

- 用于验证体内相关性和生理意义。
- 必须关注随机化、盲法、样本量、性别、年龄和伦理审批。
- 动物结果不能自动推广到人类。

### 临床样本分析

- 可增强现实相关性。
- 相关性分析不能直接证明因果。
- 需要考虑混杂因素、样本量、分层和独立队列验证。
