# 组会汇报稿模板

## 使用场景

当用户要求“汇报这篇文章 / 组会讲稿 / PPT 讲解逻辑 / 每张图怎么讲 / 老师会问什么”时使用。

---

## 汇报输出原则

- 面向非本领域听众时，先讲背景，再讲问题，再讲图。
- 不照搬论文顺序，按“问题—证据—结论”组织。
- 每张 Figure 都要有过渡语。
- 每张图只讲它对主线最重要的作用，细节放在答疑备用。
- 必须准备可能被问的问题。

---
